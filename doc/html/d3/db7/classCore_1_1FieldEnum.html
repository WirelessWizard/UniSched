Detailed Description

Field of std::string type but used for enumeration.

Simple class contains name and std::string value.

It is fully equal to Core::FieldString, but for strong type identification it is copied instead of using typedef. It has done to deny next ways of code:

 Core::FieldString string("some_field");
 Core::FieldEnum enum("some_enum", "some_value");
 string = enum;
 //---- or -------
 string = dynamic_cast<Core::FieldString>(enum);

But next way is correct:

 Core::FieldString string("some_field");
 Core::FieldEnum enum("some_enum", "some_value");
 string.value(enum.value());

Value of this class can not be empty. This class different from Core::FieldString by this.
